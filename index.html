CONCEPTO DE GIT

Git is the open source distributed version control system that facilitates GitHub activities on your laptop or desktop. This cheat sheet summarizes commonly used Git command line instructions for quick reference.

COMANDOS PRINCIPALES DE GIT

  • GIT INIT:Inicializar un proyecto en Git.
  • GIT CONFIG --GLOBAL USER.EMAIL "EMAIL": Identificar y configurar localmente el email en un repositorio local
  • GIT CONFIG --GLOBAL USER.NAME "NAME": Identificar y configurar localmente el nombre de usuario en un repositorio local
  • git config user.name: Identificar el nombre de usuario en el repositorio local
  • git config user.email: Verificar el email en el repositorio local
  • git config --global --unset-all user.name: Elimina las configuraciones del nombre usuario en el repositorio local
  • clear: Borrar la pantalla del termial
  • cd nombre del directorio: Ingresa a un directorio en la terminal
  • cd .. : Salir del directorio en la terminal
  • pwd: Mostrar la ruta del directorio actual en la terminal
  • cd: Devuelve a la ruta inicial
  • ls: Muestra la lista de contenidos de un directorio
  • git log: Mostrar el historial de cambios en el repositorio
  • git add . Agrega todos los archivos modificados al repositorio
  • git status: Verificar el estado del repositorio
  • git branch: Mostrar el nombre de la rama
  • git commit -m "Comentario"Guardar los cambios realizados en el repositorio con un mensaje resumido
  • git branch -m nombre_ramaPermite cambiar el nombre de la rama
  • git remote add origin url:Se agrega la direccion web del repositorio remoto
  • git push:Enviar los commits del repositorio local al repositorio remoto
